Prospecting Playbook
B2B best practices baked into every signal and outreach
The Core Framework
Trigger-Based Only — Only reach out when something happened. No trigger = no signal worth pursuing.
The 90-Day Window — New executives have 90 days to prove themselves. Act within 1 week of the announcement.
Multi-Thread Always — Economic buyer + champion + blocker. Reach all three independently.
Pain + Budget + Authority — A lead only converts when all three exist.
Outreach Rules
Pattern Interrupt — Never start with "I hope this finds you well." Start with an insight or observation.
Give Before You Ask — Every first touch includes a gift: insight, benchmark, or data point.
One Ask Per Touch — Never ask for a meeting AND a response. One ask only.
Follow-Ups Add Value — Never say "following up." Every touch adds something new.
Entry Point Strategy
Executive Workshop ($18-30k) — Best cold entry. Fixed scope, clear deliverable. Easy to approve.
Discovery Sprint ($15-20k) — Best for companies mid-transition. Sells itself into a retainer.
Retainer — Best sold after Sprint or Workshop proves value. Never cold. Always earned.
